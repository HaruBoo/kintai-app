# Swell 勤怠管理アプリ

🇯🇵 日本語 | [🇺🇸 English](./README.en.md)

Excelベースの勤怠管理をブラウザ・デスクトップアプリで置き換えるための勤怠管理ツールです。

## 主な機能

- **出勤・退勤の打刻**（時間はセレクトボックスで選択）
- **休憩時間の自動計算**（実働6時間超で1時間を自動控除）
- **月別データ管理**（月ごとに切り替え・保存）
- **交通費申請**（日付・区間・金額の入力と一覧管理）
- **CSVダウンロード**（Excelで開ける形式、BOM付きUTF-8）
- **土日・祝日の行色分け**（日本の祝日に対応）
- **ライト／ダーク／自動モード**切り替え
- **スマホ対応**（レスポンシブデザイン）
- **個人情報保護設計**（社員番号・氏名はメモリのみ保持、LocalStorageへは保存しない）

## 使い方

### 1. 社員番号・氏名の入力

画面上部の入力欄に社員番号と氏名を入力します。

- 社員番号はデフォルトでマスク表示（👁 ボタンで確認できます）
- 入力情報はメモリのみに保持され、**ページを閉じると消えます**
- LocalStorage・URLには保存されません

### 2. 出勤・退勤の打刻

1. 時間のセレクトボックスで打刻したい時間を選ぶ
2. **出勤** ボタンを押す
3. 退勤時に同様の手順で **退勤** ボタンを押す
4. 休憩時間と実働時間が自動で計算されます

> 打刻は当月のみ可能です。過去・未来の月は閲覧のみになります。

### 3. データの確認・修正

- テーブルの日付をクリックすると日付を編集できます
- 右端の **削除** ボタンで行を削除できます（確認あり）

### 4. 月の切り替え

画面上部の ◀ ▶ ボタンで月を切り替えます。データは月ごとに自動保存されます。

### 5. CSVダウンロード（勤怠）

テーブル右上の **ダウンロードする** ボタンを押します。

個人情報の含め方を選べます：

| 選択肢 | 内容 |
|---|---|
| 含めない（デフォルト） | 社員番号・氏名なしでダウンロード |
| 番号をマスク | 社員番号をマスク（例: `1***5`）して含める |
| 含める | 社員番号・氏名をそのまま含める |

> ダウンロード後は自動的に「含めない」に戻ります

### 6. 交通費申請

上部ナビの **交通費** タブに切り替えます。

1. 日付・出発地・到着地・金額を入力
2. **追加する** ボタンで一覧に追加
3. 月末に **ダウンロードする** でCSV出力

### 7. 表示モードの切り替え

ナビ右端のボタンで切り替えられます：

- ☀️ ライト — 常に明るいモード
- 🌓 自動 — 6時〜18時はライト、それ以外はダーク
- 🌙 ダーク — 常に暗いモード

## 技術スタック

| 項目 | 内容 |
|---|---|
| フレームワーク | React 19 + TypeScript |
| ビルドツール | Vite 7 |
| デスクトップ化 | Tauri |
| 祝日判定 | @holiday-jp/holiday_jp |
| データ保存 | localStorage（月別・PIIは除く） |

## 個人情報保護の設計方針

- 社員番号・氏名は**メモリ（Reactステート）のみ**に保持
- `localStorage` / `sessionStorage` / URLパラメータには**一切出力しない**
- PII操作は `src/services/profileService.ts` に一元化
- PII入力UIは `src/components/ProfileSection.tsx` に限定
- CSV出力は `src/components/CsvExport.tsx` で方針を管理

## ローカルで動かす

```bash
# 依存パッケージのインストール
npm install

# 開発サーバー起動（ブラウザ）
npm run dev

# デスクトップアプリとして起動（Tauri）
npx tauri dev
```
