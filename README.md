# Swell 勤怠管理アプリ

🇯🇵 日本語 | [🇺🇸 English](./README.en.md)

Excelベースの勤怠管理をブラウザ・デスクトップアプリで置き換えるための勤怠管理ツールです。

🔗 **[アプリを開く](https://kintai-app-henna.vercel.app)**

## 主な機能

- **ログイン認証**（メールアドレス＋パスワード、Supabase Auth）
- **ロールベースの画面切り替え**（管理者 / 一般従業員）
- **出勤・退勤の打刻**（時間はセレクトボックスで選択）
- **休憩時間の自動計算**（実働6時間超で1時間を自動控除）
- **月別データ管理**（月ごとに切り替え・保存）
- **交通費申請**（日付・区間・金額の入力と一覧管理）
- **領収書画像の添付**（ZIPでまとめてダウンロード）
- **CSVダウンロード**（Excelで開ける形式、BOM付きUTF-8）
- **土日・祝日の行色分け**（日本の祝日に対応）
- **ライト／ダーク／自動モード**切り替え
- **スマホ対応**（レスポンシブデザイン）
- **個人情報保護設計**（社員番号・氏名はメモリのみ保持）

### 管理者専用機能

- **ユーザー管理**（招待・ロール変更）
- **勤怠確認**（全社員分の閲覧）
- **給与明細管理**（ファイル添付・送信先メール・備考・コメント）

## 使い方

### 1. ログイン

画面上部のタブで「一般」または「管理者」を選び、メールアドレスとパスワードを入力してログインします。

| タブ選択 | アカウント種別 | 表示される画面 |
|---|---|---|
| 一般 | 一般ユーザー | 一般画面 |
| 一般 | 管理者 | 一般画面（管理者も一般画面を使える） |
| 管理者 | 管理者 | 管理者画面 |
| 管理者 | 一般ユーザー | ❌ ログイン拒否 |

> 管理者アカウントは「一般」タブからもログインできます。その場合は一般ユーザーと同じ画面が表示されます。

### 2. 出勤・退勤の打刻

1. 時間のセレクトボックスで打刻したい時間を選ぶ
2. **出勤** ボタンを押す
3. 退勤時に同様の手順で **退勤** ボタンを押す
4. 休憩時間と実働時間が自動で計算されます

> 打刻は当月のみ可能です。過去・未来の月は閲覧のみになります。

### 3. 月の切り替え

画面上部の ◀ ▶ ボタンで月を切り替えます。データは月ごとに自動保存されます。

### 4. CSVダウンロード（勤怠）

テーブル右上の **ダウンロードする** ボタンを押します。

### 5. 交通費申請

上部ナビの **交通費** タブに切り替えます。

1. 日付・出発地・到着地・金額を入力
2. 領収書画像を添付（任意）
3. **追加する** ボタンで一覧に追加
4. 月末に **ダウンロードする** でCSV＋画像をZIP出力

### 6. 表示モードの切り替え

ナビ右端のボタンで切り替えられます：

- ☀️ ライト — 常に明るいモード
- 🌓 自動 — 6時〜18時はライト、それ以外はダーク
- 🌙 ダーク — 常に暗いモード

## 個人情報保護の設計方針

- 社員番号・氏名は**メモリ（Reactステート）のみ**に保持
- `localStorage` / `sessionStorage` / URLパラメータには**一切出力しない**
- PII操作は `src/services/profileService.ts` に一元化
- PII入力UIは `src/components/ProfileSection.tsx` に限定

## ローカルで動かす

```bash
# 依存パッケージのインストール
npm install

# .env ファイルを作成して Supabase の接続情報を設定
# VITE_SUPABASE_URL=...
# VITE_SUPABASE_ANON_KEY=...

# 開発サーバー起動（ブラウザ）
npm run dev

# デスクトップアプリとして起動（Tauri）
npx tauri dev
```
